Device Setup (esp32)
This zip file contains the files needed to provision a new esp32 device. deviceSetUp.zip
See the ourProbes Videos below for a video walk through of this process.

  1. Download and open this zip file .
  2. Copy the ourProbesDeviceMaker folder to a temporary location on your machine.
  3. In a command window cd to the ourProbesDeviceMaker folder.
  4. In the probe subfolder update the config.py file (Usually by using the config.py creator in the device page, update any ssid/password info if needed.).
  5. Update the deviceDeploy.bat file to use the correct serial port.
  6. Connect the new esp32 to your pc device via a serial connection (usb cable).
    Note: you can check what serial port the device is on using the "mode" command in windows.
  7. Start the deviceDeploy.bat job.
Device Setup (Windows)
Download this zip file and expand into a new directory winOurProbe.zip
  1. Set up a device entry in ourProbes - make sure it is type "python"
  2. From the device form download its config.json file
  3. Copy the contents of the winOurProbe.zip file to a new directory on your windows machine
  4. Copy the config.json file to the same directory
  5. The device code can be run by running winOurProbe.exe - it will continue to run continuously until stopped with a ctrl-C
  6. If you plan on running this all the time, then I recommend using nssl - Non-sucking Service Manager to run winOurProbe.exe as a windows service
